  • 블로그를 왜 만들었는가?

유튜브 생활코딩에 web 편을 봤을 때 내겐 정말 충격이었고 감동적이었다. 웹 프로그래머가 되겠다고 다짐하고 책을 통해서, 그리고 다양한 유튜브 채널의 강의를 통해 html, CSS를 공부하였는데 잘 와닿지 않는 느낌이었다. html, CSS 용어들이 머릿속에 둥둥 떠다니고 제대로 이해를 못 하였다. 그런데 생활코딩의 동영상을 보고 난 후 엉망진창이었던 내 머릿속에 html, CSS 지식이 책꽂이에 정리되었다. 공부 동기를 자극하는 구성과 중간중간 web의 역사와 재밌는 이야기들은 웹에 대한 나의 흥미를 더욱 자극했다. 생활코딩 html, CSS 편 마지막에서 지금까지 배운 것들로 뭔가 나만의 사이트, 웹 페이지를 만들어 보라고 하였는데 나는 한 치의 망설임도 없이 나(Shudodog)의 개발 블로그를 만들기로 하였다. 풀 스택 웹 개발자가 되기 위해 어떤 고민을 하고 무슨 노력을 하였는지 실수를 했는지, 했다면 반성하여 더 나은 나로 발전했는지 모든 과정을 개발 블로그에 적을 것이다. 내 블로그를 본 다른 사람들이 이 블로그를 통해 좋은 정보와 긍정적인 영향을 받았으면 좋겠다. -2021.07.13
